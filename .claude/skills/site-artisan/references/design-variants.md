# Bibliothèque de variantes de design (usage unique)

Chaque variante = **palette de couleurs + typographie + ambiance**. Une variante ne doit être
utilisée que pour **un seul client**. Quand elle est attribuée, on met à jour le **tableau de
suivi** en bas du fichier (statut `🔒 utilisée`).

Les tokens correspondent aux variables CSS de `globals.css` :
`--background`, `--foreground`, `--color-cream`, `--color-cream-alt`, `--color-navy`,
`--color-navy-light`, `--color-brand`, `--color-brand-dark`.

Les polices sont chargées via `next/font/google` (remplacer la paire Geist par défaut).
